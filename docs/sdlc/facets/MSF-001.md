# MSF-001 — Clarify FR-009: explicit-marker

> Micro-spec facet: a focused enrichment of the SpecKit spec, generated for an
> identified ambiguity to give squad members the knowledge they need to execute.

**Grounded in:** FR-009

## Question (from interview)

show all models, only GA models, or only the user's recent models?

## Resolution

Show **GA models only** by default. Public-preview models are hidden behind an
explicit "Include preview models" toggle (off by default, persisted with the rest
of the selection per FR-007). This keeps the default comparison stable and billable,
and avoids surprising users with preview pricing.

## Acceptance criteria

- [ ] On first load, the comparison table lists exactly the models in `COPILOT_MODEL_OPTIONS` whose `releaseStatus === "GA"` (plus the `auto` model).
- [ ] A visible "Include preview models" toggle, default off, reveals/hides `releaseStatus === "Public preview"` rows.
- [ ] The toggle state is persisted to local storage and restored on reload (consistent with FR-007).
