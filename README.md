# Tokenizer

An original tokenizer-style progressive web app scaffold for exploring text tokenization. This project is not a clone of OpenAI or any other protected site, and it does not use copied logos, branding, source code, or assets.

## Scripts

- `npm run dev` - start the Vite dev server.
- `npm run build` - type-check and create an optimized production build.
- `npm run preview` - preview the production build locally.
- `npm run test` - run the test suite once.
- `npm run typecheck` - run TypeScript project checks.

## PWA

The app uses Vite and `vite-plugin-pwa` with a local web manifest and original SVG icon assets.
